🎮 Terminal Quiz Game

A fast-paced, colorful Python terminal quiz game packed with categories, timers, lives, grades, and a persistent leaderboard system.

✨ Features
🗂️ 4 Categories
Geography 🌍
Science 🔬
Technology 💻
Pop Culture 🎬
🎲 Random Mix Mode
⚡ 2 Difficulty Levels
Easy Mode
20 seconds/question
3 lives ❤️❤️❤️
Hard Mode
12 seconds/question
2 lives ❤️❤️
❤️ Lives System
Lose a life for:
Wrong answers
Timeouts
Game ends at 0 lives

⏱️ Animated Countdown Timer

Live terminal timer bar:
████████░░ 8s
🎯 Speed Bonus System
Faster answers = higher bonus points
📊 Performance Grades
S — Legendary
A — Excellent
B — Good
C — Average
F — Skill Issue 😅

🏆 Persistent Leaderboard

Saved in:
quiz_highscores.json
Displays Top 10 scores
👤 Player Name System
Personalized greetings and scoreboards
🎨 Beautiful ANSI Color UI
Cyan
Green
Red
Yellow
Magenta
🅰️ A/B/C/D Answer Selection
No need to type full answers
🌀 Shuffled Questions & Options
Every game feels different
📦 Requirements
Python 3.8+

No external libraries required.

▶️ Run the Game
python quiz_game.py
🕹️ Gameplay
Enter your player name
Select category
Choose difficulty

Answer using:

A / B / C / D
Beat the timer
Survive with remaining lives
Reach the leaderboard
📊 Grade System
Accuracy	Grade
95%+	S
85%+	A
70%+	B
50%+	C
Below 50%	F
